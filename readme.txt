Aplicación liberada bajo la licencia GPLv3, para el uso.

Para ejecutar depende del lenguaje de programación Python y su binding wxPython los cuales pueden descargar en las siguientes páginas web:

http://www.python.org/
http://www.wxpython.org/

Sí ejecuta desde Debian o distribuciones derivadas ejecutar en el terminal como super usuario, el siguiente comando para instalar wxPython:

aptitude install python-wxgtk2.8
